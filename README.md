# openwilma.kotlin
OpenWilma Kotlin library

## How to install

1. Add jitpack.io to root build.gradle:

```gradle
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
  ```
  
  
 2. Add dependency
 
 ```gradle
 dependencies {
	        implementation 'com.github.OpenWilma:openwilma.kotlin:Tag'
 }
 ```
 
 3. Start using OpenWilma!
